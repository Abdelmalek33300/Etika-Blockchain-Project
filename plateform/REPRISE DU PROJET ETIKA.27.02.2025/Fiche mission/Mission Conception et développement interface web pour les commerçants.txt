Mission: Conception et développement de l'interface web pour les commerçants

Module concerné: etika-merchant-portal

Contexte: Création de l'interface permettant aux commerçants de gérer leurs transactions, leurs relations avec les fournisseurs et d'interagir avec le système d'affacturage

Objectifs de la session:
1. Définir l'architecture frontend de l'application web
2. Développer les composants principaux (tableau de bord, gestion des transactions, gestion des fournisseurs)
3. Intégrer avec l'API de la plateforme
4. Implémenter la génération de QR codes pour les transactions PoP

Livrables attendus:
- Architecture et structure de code frontend (React/Vue/Angular)
- Composants principaux du tableau de bord
- Service d'intégration avec l'API
- Documentation d'utilisation

Contraintes techniques:
- Application web responsive
- Framework moderne (React recommandé)
- Accessibilité WCAG 2.1 AA

Dépendances avec autres modules:
- etika-platform-api
- etika-factoring-system (via API)
